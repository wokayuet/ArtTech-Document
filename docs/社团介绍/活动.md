# 活动记录

## 常驻活动
### 游戏沙龙
时间：每周日下午七点  
地点：T4609放映室  
报名：社团QQ群报名链接  
内容：为了保证所有参加者的参与感，目前主要玩一些电影式的剧情游戏（例如奇异人生，Her Story，暴雨等），后续也会考虑经典游戏的试玩。  
形式：人少时，手柄轮传游玩。人多时，提前确定操作人员。

### 社团招新
时间：每年九月中旬  
内容：由学校团委统一组织的“百团大战”社团招新活动。本社团将会有自己的展示摊位，摊位提供游戏试玩，同时宣传社团，吸纳新人。为后续新人立项工作的开展做准备。
