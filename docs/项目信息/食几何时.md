# 《食几何时（Hunger or Harvest）》项目介绍
《食几何时（Hunger or Harvest）》是一款基于中国古代农业知识开发的模拟经营类严肃游戏，旨在让人们通过游戏化的体验形式传播中国优秀传统文化中的农业知识。通过采用模拟经营和互动学习的体验，有助于人们更深入地理解传统农业知识的价值与应用。激发使用者对中国传统文化的认同感与自豪感。
